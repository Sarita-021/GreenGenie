# server config and APIs

### user APIs
> ***ROUTE:*** /api/v1/user
- create user : ```/new```
- get user by email : ```/:email```
- get user by firebaseUserId : ```/id/:firebaseUserId```

### blog APIs
> ***ROUTE:*** /api/v1/blog
- create blog : ```/create```
- get all blogs: ```/get/all```
- get blog by firebaseUserId : ```/get/id/:id```
- delete blog : ```/delete/:id```
